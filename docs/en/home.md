# Home (Welcome)

## Overview
Minimal start screen shown when the app opens. It does not appear in the sidebar.

## Supported inputs
- None (visual only).

## Outputs
- None.

## Workflow
- Shown on startup; selecting a module from the sidebar replaces it with the module frame.

## Technical notes
- Renders `assets/icon.png` and the current version.

## Limitations
- No actions or logic.
