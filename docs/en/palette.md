# Color Palette

## Overview
Extracts a color palette from an image (useful for UI/branding).

## Supported inputs
- Single image (`JPG/PNG/WEBP/TIFF/AVIF/HEIC/HEIF`, etc.)

## Outputs
- Color list (copyable) and palette export as an image.

## Workflow
1. Select an image.
2. Choose the number of colors.
3. Extract.
4. Copy or save.

## Common errors
- If opening fails: corrupted file or unsupported format.

## Troubleshooting
- Try converting the input to `PNG` if it fails.

## Examples
- Extract 8 colors from a screenshot.

## Technical notes
- Extraction runs in background and shows a “Processing” overlay.

## Limitations
- Results depend on content; noisy images reduce usefulness.

## Performance tips
- Use images with good lighting/contrast for cleaner palettes.
